⚫️課題3-9


①必須チェックとは何ですか？
値が入力されているかどうかのチェック（例　@NotNull）



②項目長チェックとは何ですか？
入力された文字数が指定された　”文字数or文字数以内” であるかのチェック（　例　@Size(max=10)　）



③属性チェックとは何ですか？
指定されたデータ型と一致していることをチェック（例　”@Date” で入力した値が日付の形式になっているかチェック）



④相関チェックとは何ですか？
複数の項目に入力された値の妥当性のチェック（例　@AssertTrue（message = "苗字と名前は合計10文字以内で入力してください。"）を記述しその後に
　　　　　　　　　　　　　　　　　　　　　　public boolean isValidName() {                                             
        　　　　　　　　　　　　　　　　　　　　return (firstName != null ? firstName.length() : 0)
               　　　　　　　　　　　　　　　　　　　　 + (lastName != null ? lastName.length() : 0) <= 10;   
   　　　　　　　　　　　　　　　　　　　 }のようにバリデーションについてのチェックメソッドを作る）



⑤存在チェックとは何ですか？
入力値がDBに存在しているかチェック



⑥外字チェックとは何ですか？
システムに存在しない文字かどうかチェック



⑦権限チェックとは何ですか？
入力値によるDBへのアクセス権限をチェック



⑧メールアドレスをチェックするアノテーションは何ですか？
@Email



⑨クレジットカード番号をチェックするアノテーションは何ですか？
@CreditCardNumber



⑩NULLでも空文字でもないことを確認するアノテーションは何ですか？
@NotBlank



⑪バリデーションメッセージを一括で管理するためのファイル名は何ですか？
message.properties
